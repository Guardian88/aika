Usage

Aika Networks

The Aika Networks project is a preconfigured network for the Aika algorithm. The following code snipped shows how it can be used to process an arbitrary english text. 
				
        Parser parser = new Parser();
        parser.init();

        Document doc = parser.parse("Mr. Jackson Cook was born in New York");

        for(Activation act: parser.personNamesModel.surnameCategory.getFinalActivations(doc)) {
              System.out.println(doc.getText(act.key.range));
        }

        System.out.println(doc.neuronActivationsToString(true));

        doc.clearActivations();

Named Entity Recognition / Entity Resolution example

This example demonstrates the recognition of the words 'jackson cook' as forename and surname. If these words would have occurred independently, they would have been recognized as city or profession based on the example code. The complete code for this example can be found in the class NamedEntityRecognitionTest.
Now, the first step is to create an empty model. The following map is used as a dictionary to look up the individual word input neurons. 
				
    Model m = new Model();

    HashMap<String, Neuron> inputNeurons = new HashMap<>();

    for(String word: new String[] {"jackson", "cook"}) {
        Neuron in = m.createNeuron("W-" + word);

        inputNeurons.put(word, in);
    }
These three neurons are just placeholders that will be wired up later on. 
				
    Neuron forenameCategory = m.createNeuron("C-forename");
    Neuron surnameCategory = m.createNeuron("C-surname");
    Neuron suppressingN = m.createNeuron("SUPPR");
Now, the following code snippet is the crucial section of this example. It shows how the first entity neuron is created and connected by its input synapses. The entity neurons represent the concrete meanings of the input words. As can be seen here the individual inputs have a few more properties than we are used to from other neural networks. The reason for this is that Aika not only propagates the input activation value to the next neuron but also the text range, the word position and the interpretation option. To avoid having to compute the neurons bias manually, we can use the initAndNeuron helper function for that. But in order to compute the bias it needs to know the required minimum input activation (setBias) of each input.
This entity neuron has three inputs. The first one simply checks if the word 'cook' occurred in the text. The second one is a positive feedback loop and checks if the previous word was a forename. The last one is a negative feedback loop. It makes sure that only one entity is assigned to each word.
The other entity neurons are created in a similar fashion which is therefore not shown here. 
				
    Neuron cookSurnameEntity = Neuron.init(
            m.createNeuron("E-cook (surname)"),
            3.0, // adjusts the bias
            new Synapse.Builder() // Requires the word to be recognized
                    .setNeuron(inputNeurons.get("cook"))
                    .setWeight(10.0)
                 // This input requires the input activation to have an
                 // activation value of at least 0.9
                    .setBias(-9.0)
                    .setRelativeRid(0) // references the current word
                    .setRecurrent(false)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true), // propagate the range of this input
            new Synapse.Builder() // The previous word needs to be a forename
                    .setNeuron(forenameCategory)
                    .setWeight(10.0)
                    .setBias(-9.0)
                    .setRelativeRid(-1) // references the previous word
                    .setRecurrent(true) // this input is a positive feedback loop
                    .setRangeMatch(NONE),

            // This neuron may be suppressed by the E-cook (profession) neuron, but there is no
            // self suppression taking place even though 'E-cook (surname)' is also contained
            // in suppressingN.
            new Synapse.Builder()
                    .setNeuron(suppressingN)
                    .setWeight(-20.0)
                    .setBias(0.0)
                    .setRecurrent(true) // this input is a negative feedback loop
                    .setRangeMatch(CONTAINED_IN)
    );
The category neurons in this example only have input. For a real application there would be a long list of inputs. 
				
    Neuron.init(forenameCategory, 0.0,
            new Synapse.Builder() // In this example, only one forename is considered.
                    .setNeuron(jacksonForenameEntity)
                    .setWeight(10.0)
                    .setBias(0.0)
                    .setRelativeRid(0)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true)
    );
    Neuron.init(surnameCategory, 0.0,
            new Synapse.Builder()
                    .setNeuron(cookSurnameEntity)
                    .setWeight(10.0)
                    .setBias(0.0)
                    .setRelativeRid(0)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true)
    );
The last neuron is the suppressing neuron. It is used as a mediator in order to avoid having to connect all of these neurons with each other. Self suppression does not take place. 
				
    Neuron.init(suppressingN, 0.0,
            new Synapse.Builder()
                    .setNeuron(cookProfessionEntity)
                    .setWeight(10.0)
                    .setBias(0.0)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true),
            new Synapse.Builder()
                    .setNeuron(cookSurnameEntity)
                    .setWeight(10.0)
                    .setBias(0.0)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true),
            new Synapse.Builder()
                    .setNeuron(jacksonCityEntity)
                    .setWeight(10.0)
                    .setBias(0.0)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true),
            new Synapse.Builder()
                    .setNeuron(jacksonForenameEntity)
                    .setWeight(10.0)
                    .setBias(0.0)
                    .setRangeMatch(EQUALS)
                    .setRangeOutput(true)
    );
Now that the model is complete, we can start processing an actual text. 
				
    Document doc = m.createDocument("mr. jackson cook was born in new york ");

    int i = 0;
    int wordPos = 0;
    for(String w: doc.getContent().split(" ")) {
        int j = i + w.length();

        inputNeurons.get(w).addInput(doc,
                    new Activation.Builder()
                        .setRange(i, j)
                        .setRelationalId(wordPos)
                    );

        i = j + 1;
        wordPos++;
    }

    // Search for the best interpretation of this text.
    doc.process();
                
			
				
    System.out.println(doc.activationsToString());
    System.out.println();

    System.out.println("Final interpretation: " + doc.bestInterpretation.toString());

    doc.clearActivations();

     *     |----------------------------|
     *     | ------                     |
     *     -*| &  |                     |
     *  A ---| PA |------\   -------    |
     *       ------       \  |     |    |
     *                     \-| OR  |    |
     *       ------          |     |    |
     *  B ---| &  |----------|Suppr|----|
     *     -*| PB |         -|     |    |
     *     | ------        / -------    |
     *     |----------------------------|
     *                   /              |
     *       ------     /               |
     *  C ---| &  |----/                |
     *     -*| PC |                     |
     *     | ------                     |
     *     |----------------------------|
     */

    @Test
    public void testMutualExclusion() {
        Model m = new Model();

        // Create the input neurons for the network.
        Neuron inA = m.createNeuron("IN-A");
        Neuron inB = m.createNeuron("IN-B");
        Neuron inC = m.createNeuron("IN-C");

        // Instantiate the suppressing neuron. Its inputs will be added later on.
        Neuron pSuppr = m.createNeuron("SUPPRESS");

        // Create three neurons that might be suppressed by the suppressing neuron.
        // createAndNeuron is a helper method which is used set the synapse weights
        // and bias of new neuron.
        Neuron pA = Neuron.init(
                m.createNeuron("A"),
                3.0,
                new Synapse.Builder()
                        .setNeuron(inA)
                        .setWeight(10.0)
                        .setBias(-10.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true),
                new Synapse.Builder()                     // This input is negated
                        .setNeuron(pSuppr)
                        .setWeight(-10.0)
                        .setBias(0.0)
                        .setRecurrent(true)
                        .setRangeMatch(CONTAINED_IN)
        );

        Neuron pB = Neuron.init(
                m.createNeuron("B"),
                5.0,
                new Synapse.Builder()
                        .setNeuron(inB)
                        .setWeight(10.0)
                        .setBias(-10.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true),
                new Synapse.Builder()
                        .setNeuron(pSuppr)
                        .setWeight(-10.0)
                        .setBias(0.0)
                        .setRecurrent(true)
                        .setRangeMatch(CONTAINED_IN)
        );

        Neuron pC = Neuron.init(
                m.createNeuron("C"),
                2.0,
                new Synapse.Builder()
                        .setNeuron(inC)
                        .setWeight(10.0)
                        .setBias(-10.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true),
                new Synapse.Builder()
                        .setNeuron(pSuppr)
                        .setWeight(-10.0)
                        .setBias(0.0)
                        .setRecurrent(true)
                        .setRangeMatch(CONTAINED_IN)
        );

        // Finally we add all the inputs to the suppressing neuron.
        Neuron.init(
                pSuppr,
                0.0,
                new Synapse.Builder()
                        .setNeuron(pA)
                        .setWeight(10.0)
                        .setBias(0.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true),
                new Synapse.Builder()
                        .setNeuron(pB)
                        .setWeight(10.0)
                        .setBias(0.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true),
                new Synapse.Builder()
                        .setNeuron(pC)
                        .setWeight(10.0)
                        .setBias(0.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true)
        );

        Neuron outN = Neuron.init(new Neuron("OUT"), 0.0,
                new Synapse.Builder()
                        .setNeuron(pB)
                        .setWeight(1.0)
                        .setBias(0.0)
                        .setRecurrent(false)
                        .setRangeMatch(EQUALS)
                        .setRangeOutput(true)
        );

        // Now that the model is complete, apply it to a document.

        Document doc = m.createDocument("foobar");

        // Add input activations starting at char 0 and ending at char 1
        // These inputs will be immediately propagated through the network.
        inA.addInput(doc, 0, 1);
        inB.addInput(doc, 0, 1);
        inC.addInput(doc, 0, 1);

        // Computes the selected option
        doc.process();

        System.out.println("Final Interpretation: " + doc.bestInterpretation);
        System.out.println();

        System.out.println("Show all conflicts with the selected option:");
        for(Conflict c: doc.selectedOption.conflicts.primary.values()) {
            System.out.println(c.conflict);
        }
        System.out.println();

        System.out.println("Output activation:");
        for(Activation act: outN.node.getActivations(t)) {
            System.out.println("Text Range: " + act.key.range);
            System.out.println("Option: " + act.key.interpretation);
            System.out.println("Node: " + act.key.node);
            System.out.println("Rid: " + act.key.rid);
            System.out.println("Activation weight: " + act.finalState.value);
            System.out.println();
        }

        System.out.println("All activations:");
        System.out.println(doc.activationsToString());
        System.out.println();

        System.out.println("Selected activations:");
        System.out.println(doc.activationsToString(false, true));

        doc.clearActivations();
    }

Pattern matching example

				
    @Test
    public void testPatternMatching() {
        Model m = new Model();

        Map<Character, Neuron> inputNeurons = new HashMap<>();

        // Create an input neuron for every letter in this example.
        for(char c: new char[] {'a', 'b', 'c', 'd', 'e', 'f'}) {
            Neuron in = m.createNeuron(c + "");

            inputNeurons.put(c, in);
        }


        // Create a pattern neuron with relational neurons as input. The numbers that are
        // given in the inputs are the relational ids (relativeRid) which specify the word positions
        // of the inputs relative to each other.
        Neuron pattern = Neuron.init(
                m.createNeuron("BCDE"),
                0.4,
                new Synapse.Builder()
                        .setNeuron(inputNeurons.get('b'))
                        .setWeight(1.0)
                        .setBias(-0.9)
                        .setRecurrent(false)
                        .setRelativeRid(0)
                        .setStartRangeMatch(Operator.EQUALS)
                        .setEndRangeMatch(Operator.GREATER_THAN)
                        .setStartRangeOutput(true),
                new Synapse.Builder()
                        .setNeuron(inputNeurons.get('c'))
                        .setWeight(1.0)
                        .setBias(-0.9)
                        .setRecurrent(false)
                        .setRelativeRid(1)
                        .setRangeMatch(RangeRelation.CONTAINS),
                new Synapse.Builder()
                        .setNeuron(inputNeurons.get('d'))
                        .setWeight(1.0)
                        .setBias(-0.9)
                        .setRecurrent(false)
                        .setRelativeRid(2)
                        .setRangeMatch(RangeRelation.CONTAINS),
                new Synapse.Builder()
                        .setNeuron(inputNeurons.get('e'))
                        .setWeight(1.0)
                        .setBias(-0.9)
                        .setRecurrent(false)
                        .setRelativeRid(3)
                        .setStartRangeMatch(Operator.LESS_THAN)
                        .setEndRangeMatch(Operator.EQUALS)
                        .setEndRangeOutput(true)
        );

        // Create a simple text document.
        Document doc = m.createDocument("a b c d e ");


        // Then add the characters
        int wordPos = 0;
        for(int i = 0; i < doc.length(); i++) {
            char c = doc.getContent().charAt(i);
            if(c != ' ') {
                inputNeurons.get(c).addInput(doc,
                    new Activation.Builder()
                        .setRange(i, i + 1)
                        .setRelationalId(wordPos)
                );
            } else {
                wordPos++;
            }
        }

        // Computes the final interpretation
        doc.process();

        System.out.println("Output activation:");
        for(Activation act: pattern.node.getActivations(t)) {
            System.out.println("Text Range: " + act.key.range);
            System.out.println("Option: " + act.key.interpretation);
            System.out.println("Node: " + act.key.node);
            System.out.println("Rid: " + act.key.rid);
            System.out.println();
        }

        System.out.println("All activations:");
        System.out.println(doc.activationsToString(false, true));
        System.out.println();

        doc.clearActivations();
    }
