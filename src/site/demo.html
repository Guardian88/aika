Demonstrator

Using this demonstrator you can test what happens if an Aika network is applied to an arbitrary text. The network that is used here consists of neurons representing forenames, surnames, cities, countries, colors, weekdays, months, common english words and a few grammar rules. The sources for this example network can be found on Github. The prefixes of the neuron labels can be interpreted as follows: "E-": Entity (Word meaning), "W-": word, "P-": Phrase and "C-": Category
